# NO BRAKE: Turbo Rush - 技术规格

## 1. 引擎与工具

| 类别 | 选择 |
|------|------|
| 游戏引擎 | Godot 4.2+ |
| 美术工具 | Aseprite |
| 音频工具 | Bfxr, Bosca Ceoil |
| 版本控制 | Git |

## 2. 核心物理参数

### 2.1 车辆参数

```
最大速度: 400 px/s
加速度: 200 px/s²
转向速度: 180°/s (静止) → 90°/s (最大速度)
摩擦系数: 0.98 (地面)
碰撞弹性: 0.3
```

### 2.2 氮气系统

```
最大能量: 100
能量恢复: 5/秒
加速倍率: 1.5x
持续时间: 2秒
```

## 3. 关卡规格

```
单关尺寸: 1280x720 (可视区域)
Tile大小: 16x16
最大关卡: 80x45 tiles
检查点: 每关 2-4 个
```

| 指标 | 目标 |

## 6. 控制器支持

| 控制器 | 按键映射 |
|--------|----------|
| Xbox | A=加速, B=刹车, X=氮气, Y=重开 |
| PlayStation | ×=加速, ○=刹车, □=氮气, △=重开 |
| 通用 | Button0=加速, Button1=刹车 |
